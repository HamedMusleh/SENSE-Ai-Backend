# SENSE Triage Labels - Quick Reference

This file is a quick reference only.
The full labeling authority is `annotation_guide.md`.

## Labels

1. Safe / Regulated
- Normal child behavior
- Play, curiosity, ordinary complaints
- No clear distress signal

2. Distressed / Needs Support
- Fear, sadness, anxiety, nightmares, withdrawal, somatic complaints
- Child is still communicating
- Default when symptomatic but non-urgent

3. High Risk / Urgent
- Explicit or strongly implied urgent signal
- First-person death reference
- Death wish
- Panic with explicit fear of dying
- Depersonalization or severe shutdown
- Must be routed to human specialist

4. Unclear / Need More Context
- Silence
- One-word answer
- Vague or ambiguous response
- Not automatically Safe

## Priority Rules

- Self-directed death reference → High Risk
- Panic with explicit death fear → High Risk
- Death of others without self-directed wish → Distressed
- Fear/anxiety/body complaint → Distressed
- Silence/single-word response → Unclear
- Play/curiosity/ordinary complaint → Safe

## Conservative Defaults

- When unsure between Distressed and High Risk → Distressed
- When unsure between Safe and Distressed → Distressed
- Do not overuse High Risk
